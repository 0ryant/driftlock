# Driftlock Maintainer Skill

## Purpose

Maintain lane policy, graph freshness, claims, overrides, and release governance.

## Rules

- Prefer narrowing task scope over broad override.
- Every override must be auditable.
- Stale base refs invalidate readiness.
- Lane changes require governance review.
- Contract changes require ADR review.

## Override record

Include task, actor, reason, affected resources, expiry, and reviewer.
